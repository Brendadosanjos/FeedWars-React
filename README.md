# React + TypeScript + Vite Project

Este projeto é uma aplicação simples que simula um feed de publicações de usuários em uma rede social. 

## Tecnologias Utilizadas

- `TypeScript`: Linguagem de programação que adiciona tipagem estática ao JavaScript.
- `Vite`: Ferramenta de build e desenvolvimento rápida e moderna.
- `Tailwind CSS`: Framework utilitário de CSS para criar layouts responsivos e estilizados rapidamente.
- `JSON Server`: Para simular uma API RESTful com os dados do `data.json`.
- `Radix UI`: Conjunto de componentes acessíveis e de fácil personalização.
- `JavaScript (ES6+)`: Baseado em JavaScript, mas com a tipagem estática oferecida pelo TypeScript.
